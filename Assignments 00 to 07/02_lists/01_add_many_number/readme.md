# ➕ Sum of Numbers

## 📌 Description
This Python program takes a list of numbers entered by the user and calculates their sum.

## 🔥 Features
- Accepts multiple numbers as input.
- Uses a function to calculate the sum.
- Displays the total sum in a formatted output.

## 🚀 Installation & Usage

### CLI Version
1. Ensure Python is installed.
2. Save the script as `sum_of_numbers.py`.
3. Run the script using:
   ```sh
   python sum_of_numbers.py
   ```
4. Enter numbers separated by spaces when prompted.

### 🖥️ Example Output
```
Enter numbers separated by spaces: 3 5 7 2
The sum of the numbers is: 17
```
